# 🏠 Machine Learning Based Buyer Segmentation & Investment Profiling for Real Estate Market Intelligence

> An end-to-end Machine Learning project that analyzes real estate buyer behavior using clustering techniques and provides interactive business insights through a Streamlit dashboard.

---

## 📌 Project Overview

The real estate industry generates large volumes of customer data, making it difficult to identify different buyer profiles manually. This project applies **Machine Learning-based clustering** to segment buyers according to their demographic characteristics, purchasing behavior, investment objectives, and satisfaction levels.

The developed Streamlit dashboard enables real estate organizations to analyze customer segments, understand investment patterns, and support data-driven marketing strategies.

---

## 🎯 Objectives

- Segment real estate buyers using Machine Learning.
- Analyze customer investment behavior.
- Identify demographic and geographic buying patterns.
- Generate business insights for targeted marketing.
- Build an interactive dashboard for decision-making.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sravya200619/Buyer_Segmentation_Project.git
```

Move into the project directory

```bash
cd Buyer_Segmentation_Project
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
